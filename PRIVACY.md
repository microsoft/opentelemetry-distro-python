# Privacy

This repository contains source code for a Python package intended to configure observability pipelines.

The repository itself is not intended to collect personal data simply by being cloned or reviewed. However, software built from this repository may be configured to collect and export telemetry such as traces, metrics, logs, and application metadata to Microsoft services or third-party backends.

If you use this software to collect telemetry from users or systems, you are responsible for:

- complying with applicable privacy, data protection, and notice requirements
- configuring only the telemetry you intend to collect
- choosing appropriate exporters and retention settings
- providing any required notices to end users

Microsoft's privacy statement is available at https://privacy.microsoft.com/privacystatement.
