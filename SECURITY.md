# Security

Microsoft takes the security of our software products and services seriously, which includes all source code repositories managed through our GitHub organizations.

## Reporting Security Issues

Do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in this repository that meets [Microsoft's definition of a security vulnerability](https://aka.ms/opensource/security/definition), report it to the Microsoft Security Response Center (MSRC):

- Report online: https://aka.ms/opensource/security/create-report
- Email: [secure@microsoft.com](mailto:secure@microsoft.com)

If possible, include:

- type of issue
- affected source files or paths
- branch, commit, or version information
- reproduction steps
- impact assessment
- proof of concept if available

Additional information is available at https://aka.ms/opensource/security/msrc.

## Preferred Language

We prefer all communications to be in English.

## Disclosure Policy

Microsoft follows the principle of coordinated vulnerability disclosure.
